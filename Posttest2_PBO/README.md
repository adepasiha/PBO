## KOnsep OOP yang digunakan 
1. EScaptulation     
Atribut dalam class dibuat private sehingga tidak dapat diakses langsung dari luar class 


2. Getter dan Setter     
Untuk mengakses dan mengubah nilai atribut digunakan method:


-Getter

-setter


contoh:


public String getMerk() { return merk; } public void setMerk(String merk) { this.merk = merk; }


## 3. Access Modifier

Access modifier yang digunakan  dalam program ini:


-Private -> untuk atribut pada class

-Public -> untuk method
